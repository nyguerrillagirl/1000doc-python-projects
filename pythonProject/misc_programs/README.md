# Files in this directory:

- README.md : This file provides an overview of the contents of the directory, describing the purpose of each file and how they relate to each other. It serves as a guide for users to understand the functionality of the code and how to use it effectively.
- client.py + server.py : These files contain the implementation of a simple client-server communication using sockets in Python. The server listens for incoming connections and responds to client requests, while the client connects to the server and sends messages. It is intended to illustrate struct package.
- working_with_struct.py : This file demonstrates how to use the struct package in Python to convert between Python values and C structs represented as Python bytes objects. It includes examples of packing and unpacking data using struct, which is useful for handling binary data in network communication or file I/O.
